HEADLINE

Селекторы используются для форматирования разных свойств документа, в том числе цветов, размеров и т.д. Виды: по тегам (применяется только к Body), по классам - для отображения нескольких элементов на странице, используют атрибут class и знак ".", по идентификаторам - для отображения конкретного элемента с атрибутом id и знаком "#", универсальные для установления стиля одновременно всем элементам и обозначается "*", вложенные селекторы (обозначаются пробелом), дочерние -внутри родительского элемента (со знаком >), соседние - элементы друг за другом (со знаком "+"), родственные - все близлежащие элементы (обозначаются знаком ~) ...
